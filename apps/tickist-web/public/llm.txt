# Tickist

> Tickist is an open-source task and project workspace built with Angular and Supabase. It helps people collect work in an Inbox, organize it into projects, plan deadlines and reminders, and focus on the next useful action.

## Product summary

Tickist provides an Inbox, nested projects, tasks, tags, subtasks, recurring work, reminders, collaboration, dashboards, statistics, notifications, backup/restore, and API tokens. It is a private workspace application: project, task, tag, team, settings, and API routes require an authenticated user.

## Public web surface

- Canonical public pages: `https://tickist.com/`, `https://tickist.com/en/blog`, and `https://tickist.com/pl/blog`.
- The public blog is responsive, requires no sign-in, and has no posts, comments, CMS, or admin panel in its first release.
- Blog content is code-reviewed repository content in `apps/tickist-web/src/app/features/blog/blog-content.ts`. English and Polish have independent post/category/tag registries; do not infer translations or link records automatically.
- The Cloudflare Worker supplies canonical, robots, Open Graph, Twitter-card, and JSON-LD metadata for each blog language index before Angular runs.
- Sitemap: `https://tickist.com/sitemap.xml`
- Crawler policy: `https://tickist.com/robots.txt`
- Authentication and user workspace routes are intentionally not public content.

## Technology

- Angular 21 standalone application with signals, Vite, Tailwind CSS, and DaisyUI
- Supabase Postgres, Auth, Storage, Realtime, Row Level Security, and Edge Functions
- Nx workspace with Vitest and Playwright
- Cloudflare Worker deployment and AWS SES email delivery
